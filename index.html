Terminalo Komandos

  • ls/ dir - parodo aplankalus ir failus dabartineje direktorijoje
  • cd - changeDirectory - keičia didrektorija
  • TAB - just do it - skirtas užpildyti terminalo komandą arba pateikti pasiūlimus
  • CTRL C -atšaukia vygdoma veiksmą/ nutraukia komandą
  • vidurinis/dešinys mygtukas pelės - iklijuoja tekstą į komandinę eilutę
  • cd .. - žingsnis atgal
  • ARROW UP/DOWN - vaikščiojimas per terminalo istoriją

Git Komandos

  • git init - inicializuoja git repozitorija
  • git status - parodo git repozitorijos statusą
  • git add ./*/ - pridėti failus prie paruoštų commintinimui failų
  • git commit -m "zinute" - sukuriama kodo versija su žinute
  • git restore ./*/file - panaikina visus šios kodo versijos pakeitimus (kurie nera addinti)
  • git restore -- staged ./*/file - išima nurodytą failą/us iš staged zonos
  • git push - nusiunčia nuajus lokalius commitus
  • git log - rod git commitu istorija
